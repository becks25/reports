Instructions

Submitting an Infraction

All infractions must have a staff member and an infraction selected.

Staff: Start typing. When names of employees pop up, select the correct one. Only one employee can be selected per infraction.

Infraction: Select desired infraction from the dropdown menu.

Submitting an Incident

Incidents require staff name(s), time of incident, and a written report.

Staff: Start typing a name. Select the desired staff member and click add to include them in the report. You can add as many staff members as you need to.

Time: This defaults to the current time. Enter the time when the incident occurred.

Report: Enter a description of what happened.

Action: If any disciplinary action was taken, enter it here.

Cops called: Select yes or no.

My Reports

You can see all reports you have submitted. If a report has been submitted in the last 12 hours, you can make necessary changes to it.
